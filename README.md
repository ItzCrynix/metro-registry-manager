# Metro Registry Manager

Um sistema que controla arquivos binários que guardam informações a respeito do sistema de metrôs da cidade de São Paulo.

---

## Como Rodar o Código

### Pré-requisitos

Antes de começar, você vai precisar ter instalado em sua máquina:
* Compilador de código c/c++
* Make

### Execução

1. **Clone o repositório:**

```bash
git clone https://github.com/ItzCrynix/metro-registry-manager.git
```

2. **Entre no diretório:**
```bash
cd metro-registry-manager
```

3. **Execute a aplicação:**
```bash
make run
```
